Keyboard Shortcuts

Default Shortcuts:

  • Ctrl+Shift+P - Toggle teleprompter overlay
  • Ctrl+Shift+R - Start/Stop recording mode
  • Space - Pause/Resume scrolling (when overlay active)
  • Escape - Close teleprompter overlay
  • ↑ / ↓ - Adjust scroll speed

Note: You can change these shortcuts in Chrome's extension keyboard shortcuts settings.
